# Phase 1 Strategy Blueprint (BrainoGym Redesign)

## 1) Business Goals
- Primary: increase qualified enquiries for demo/trial classes.
- Secondary: improve trust, authority, and local discoverability.
- Tertiary: create scalable npm-based website architecture for future growth pages.

## 2) Success Metrics (Phase 1 targets)
- Clear CTA visibility in every major section.
- Zero broken internal links/assets from existing content.
- Complete content map for homepage + supporting pages.
- All existing media assets mapped to planned usage.

## 3) Planned Information Architecture
- `/` Home (primary conversion landing page)
- `/about` Founder story + teaching philosophy + educator trust
- `/programs` Program overview (Abacus, Vedic Maths, Robotics, Personality)
- `/results` Student outcomes + scanner results + achievement proof
- `/gallery` Structured media gallery (events, classes, scanner)
- `/contact` Lead capture + location + direct call/WhatsApp
- Future additions: `/testimonials`, `/faq`, `/blog`, `/resources`

## 4) Homepage Section Blueprint (Final)
1. Header + sticky action bar
2. Hero (benefit-led headline + dual CTA + trust strip)
3. Credibility metrics (replace placeholders with real numbers)
4. Program outcome cards (not just course names)
5. Founder trust block
6. Social proof (testimonials + parent quotes + result highlights)
7. Event/media spotlight (SCANNER + class moments)
8. Why BrainoGym method (differentiators)
9. FAQ mini-section (high-intent objections)
10. Final high-conversion CTA footer band

## 5) Conversion Strategy
- Primary CTA text: `Book Free Demo Class`
- Secondary CTA text: `Talk to Counselor`
- Tertiary CTA text: `View Results & Success Stories`
- CTA placement: hero, after metrics, after programs, before footer.
- Offer hooks: trial class, program guidance call, worksheet/resource lead magnet.

## 6) SEO and Content Strategy (Planned in design)
- Unique page meta title + description per route.
- Local intent keywords (Delhi + kids math + abacus + vedic maths + robotics).
- Structured heading hierarchy (`h1` single, semantic `h2/h3` flow).
- Schema plan: `EducationalOrganization`, `LocalBusiness`, `Course`, `FAQPage`.
- Internal linking plan: Home -> Programs -> Results -> Contact conversion loop.

## 7) UX/Visual Direction (Golden Professional Theme)
- Palette direction: premium gold + warm neutrals + high-contrast text.
- Tone: professional institute, not template-like coaching site.
- Visual hierarchy: large outcomes, tight copy blocks, clean whitespace rhythm.
- Mobile-first: touch-friendly CTA sizes, clear section segmentation, fast scanning.

## 8) Known Baseline Issues to Fix During Build
- Broken/missing asset references (`team-member-placeholder.jpg`, malformed links).
- Inconsistent favicon path casing.
- Inline CSS/JS duplication across all pages.
- Missing SEO metadata and social metadata.
- Several pages without ideal heading/semantic consistency.

## 9) Phase 1 Output Status
- Architecture defined.
- Conversion-first homepage layout defined.
- SEO/content direction defined.
- Asset planning delegated to `ASSET_USAGE_MATRIX.md`.
- Copy framework delegated to `HOMEPAGE_COPY_FRAMEWORK.md`.
